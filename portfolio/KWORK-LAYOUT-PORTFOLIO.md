---

## Как загрузить на Kwork

1. Откройте кворк в рубрике «Доработка и адаптация верстки»
2. Раздел **Портфолио** → **Добавить работу**
3. Загрузите PNG из `portfolio/screenshots/kwork-layout/`
4. Скопируйте **название** и **описание** из таблицы ниже
5. В конец описания добавьте ссылку на живое демо

### Пересборка скриншотов

```bash
cd portfolio
node export-layout-screenshots.mjs
```

## Тексты для каждой работы

### 1. Адаптация вёрстки До/После ⭐

**Название:**  
Адаптация вёрстки сайта — интерактивное сравнение До/После

**Описание:**  
Доработка вёрстки интернет-магазина: убрал горизонтальную прокрутку, исправил переполнение карточек и заголовков. Интерактивное демо с переключением «до / после» и эмуляцией ширины экрана (desktop, tablet, mobile).

**Стек:** HTML, CSS, Flexbox, медиазапросы  
**Тип:** учебный проект / демо  
**Живое демо:** https://kirikos37.github.io/test/projects/07-responsive-adapt.html

---

### 2. Адаптивный каталог товаров

**Название:**  
Адаптивный каталог интернет-магазина — CSS Grid

**Описание:**  
Сетка каталога 3→2→1 колонка, боковые фильтры превращаются в чипы на мобильных. Переключение вида «плитка / список». Показывает умение адаптировать сложные макеты e-commerce.

**Стек:** HTML, CSS Grid, Flexbox  
**Тип:** учебный проект / демо  
**Живое демо:** https://kirikos37.github.io/test/projects/08-catalog-grid.html

---

### 3. Адаптивная навигация

**Название:**  
Доработка шапки и меню под mobile / tablet / desktop

**Описание:**  
Адаптация навигации: полное меню на desktop, компактное на tablet, бургер-меню на mobile. Sticky header, корректные брейкпоинты. Типичная задача при доработке готовых сайтов.

**Стек:** HTML, CSS, адаптив  
**Тип:** учебный проект / демо  
**Живое демо:** https://kirikos37.github.io/test/projects/09-mobile-nav.html

---

### 4. Адаптивная форма заказа

**Название:**  
Адаптивная форма с валидацией — 2 колонки → 1 на mobile

**Описание:**  
Форма заявки: поля в две колонки на desktop, в столбик на мобильных. Валидация email, фокус-состояния, крупные touch-зоны (44px+). Сообщение об успешной отправке.

**Стек:** HTML, CSS, JavaScript  
**Тип:** учебный проект / демо  
**Живое демо:** https://kirikos37.github.io/test/projects/10-form-responsive.html

---

### 5. Таблица тарифов → карточки

**Название:**  
Адаптивная таблица тарифов без горизонтального скролла

**Описание:**  
Таблица цен на desktop автоматически превращается в карточки на экранах &lt;700px. Чистый CSS, без JavaScript. Классическое решение для прайс-листов и сравнительных таблиц.

**Стек:** HTML, CSS, @media  
**Тип:** учебный проект / демо  
**Живое демо:** https://kirikos37.github.io/test/projects/11-table-adaptive.html
