**Описание:**  
Лендинг для frontend/React-разработчика: hero-блок, живой счётчик (интерактив без перезагрузки), табы с компонентами, CTA. Тёмная тема, оранжевый акцент React. Показывает умение делать **не статичные страницы, а живые интерфейсы**.

## Ссылка на демо в описании кворка

### Честность
Указывайте **«учебный проект / демо»** — это нормально для начинающего. Покупатели ценят честность и видят реальный уровень.

### Названия
Пишите **что получит клиент**, а не только технологию:
- ❌ «HTML/CSS сайт»
- ✅ «Лендинг для кафе с меню и онлайн-записью»

### Первые 3 работы = ваш профиль
На Kwork первые карточки видят чаще всего. Ставьте React и портфолио первыми — это совпадает с вашим кворком.

Вставьте в описание кворка:

```
Живые демо: https://kirikos37.github.io/test/
```
