## 🔧 Переменные

### Что такое переменная?
Переменная - это как коробочка с названием, в которой можно хранить разные данные (текст, числа, даты и т.д.). У каждой коробочки есть имя, по которому мы можем к ней обращаться.

### Способы объявления переменных

#### 1. `let` - изменяемая переменная
```javascript
let name = "Иван";
name = "Петр"; // можно изменить значение
console.log(name); // выведет: Петр
```

**Когда использовать:**
- Когда значение может изменяться
- Для счетчиков, результатов вычислений
- Для данных, которые обновляются

**Примеры:**
```javascript
let score = 0;
score = score + 10; // увеличиваем счет
score += 5; // то же самое, но короче

let currentUser = "гость";
currentUser = "admin"; // меняем пользователя
```

#### 2. `const` - константа (неизменяемая)
```javascript
const PI = 3.14159;
// PI = 3.14; // ОШИБКА! Нельзя изменить константу
```

**Когда использовать:**
- Для значений, которые не изменяются
- Для математических констант
- Для настроек приложения
- Для ссылок на объекты (сам объект можно изменять, но ссылку - нет)

**Примеры:**
```javascript
const MAX_USERS = 100;
const APP_NAME = "Моё приложение";
const BIRTH_YEAR = 1990;

// Объект можно изменять, но ссылку - нет
const user = { name: "Иван", age: 25 };
user.age = 26; // ✅ Можно
// user = { name: "Петр" }; // ❌ Ошибка!
```

#### 3. `var` - старый способ (НЕ РЕКОМЕНДУЕТСЯ)
```javascript
var oldVariable = "старое значение";
```

**Почему не использовать:**
- Устаревший синтаксис
- Проблемы с областью видимости
- Может создавать неожиданные ошибки

### Правила именования переменных

#### ✅ Что можно:
```javascript
let userName = "Иван";        // camelCase (рекомендуется)
let user_name = "Петр";      // snake_case
let UserName = "Мария";      // PascalCase (для классов)
let $price = 100;            // начинается с $
let _private = "секрет";     // начинается с _
let name123 = "тест";        // цифры в конце
```

#### ❌ Что нельзя:
```javascript
let 123name = "тест";        // начинать с цифры
let user-name = "тест";      // дефис в имени
let let = "тест";            // ключевые слова
let "name" = "тест";         // кавычки в имени
```

## 📊 Типы данных

### Строки (текст)
```javascript
let firstName = "Анна";
let lastName = 'Петрова';           // одинарные кавычки тоже работают
let fullName = `${firstName} ${lastName}`; // шаблонные строки
let message = "Привет, мир!";
```

### Числа
```javascript
let age = 25;
let temperature = -5.5;
let population = 1000000;
let pi = 3.14159;
```

### Булевы значения (true/false)
```javascript
let isLoggedIn = true;
let hasPermission = false;
let isActive = true;
```

### Undefined и null
```javascript
let notDefined;           // undefined (не задано значение)
let emptyValue = null;    // null (намеренно пустое значение)
```

## ➕ Операторы

### Математические операторы
```javascript
let a = 10;
let b = 3;

console.log("Сложение:", a + b);        // 13
console.log("Вычитание:", a - b);       // 7
console.log("Умножение:", a * b);       // 30
console.log("Деление:", a / b);         // 3.333...
console.log("Остаток от деления:", a % b); // 1
console.log("Возведение в степень:", a ** b); // 1000
```

### Операторы сравнения
```javascript
console.log("a > b:", a > b);           // true
console.log("a < b:", a < b);           // false
console.log("a == b:", a == b);         // false
console.log("a === b:", a === b);       // false (строгое сравнение)
console.log("a != b:", a != b);         // true
console.log("a !== b:", a !== b);       // true
```

### Логические операторы
```javascript
let x = true;
let y = false;

console.log("И (&&):", x && y);         // false
console.log("ИЛИ (||):", x || y);       // true
console.log("НЕ (!):", !x);             // false
```

## 🔀 Условные конструкции

### Простое условие if-else
```javascript
let score = 85;

if (score >= 90) {
    console.log("Отлично! Оценка A");
} else if (score >= 80) {
    console.log("Хорошо! Оценка B");
} else if (score >= 70) {
    console.log("Удовлетворительно! Оценка C");
} else {
    console.log("Неудовлетворительно! Оценка F");
}
```

### Тернарный оператор
```javascript
let result = score >= 60 ? "Сдал" : "Не сдал";
console.log("Результат:", result);
```

### Switch для множественных условий
```javascript
let day = "понедельник";
switch (day) {
    case "понедельник":
        console.log("Начинается рабочая неделя");
        break;
    case "пятница":
        console.log("Ура! Пятница!");
        break;
    case "суббота":
    case "воскресенье":
        console.log("Выходные!");
        break;
    default:
        console.log("Обычный рабочий день");
}
```

## ⚙️ Функции

### Обычная функция
```javascript
function greet(name) {
    return `Привет, ${name}!`;
}

// Вызов функции
console.log(greet("Мария"));
```

### Функция с несколькими параметрами
```javascript
function add(a, b) {
    return a + b;
}

console.log("Сумма 5 + 3 =", add(5, 3));
```

### Стрелочная функция (современный синтаксис)
```javascript
const multiply = (x, y) => x * y;
console.log("Произведение 4 * 6 =", multiply(4, 6));
```

### Функция без параметров
```javascript
const getRandomNumber = () => Math.floor(Math.random() * 100);
console.log("Случайное число:", getRandomNumber());
```

### Простая игра
```javascript

let computerScore = 0;
let round = 1;

function playRound() {
    let playerChoice = "камень";
    let computerChoice = "ножницы";
    
    if (playerChoice === computerChoice) {
```

## 💡 Важные советы

### Основные принципы
1. **Всегда объявляй переменные** перед использованием
2. **Используй понятные имена** - `userName` лучше чем `u`
3. **Выбирай правильный тип:**
   - `let` - для изменяемых значений
   - `const` - для неизменяемых
4. **Не используй `var`** - это устаревший синтаксис
5. **Используй camelCase** для имен переменных

### Лучшие практики
- Всегда используй `===` для сравнения вместо `==`
- Пиши понятные имена переменных и функций
- Не забывай про точки с запятой в конце строк
- Комментируй свой код
- Используй современный синтаксис ES6+

## ❓ Частые вопросы

**В: Можно ли изменить `const` переменную?**
О: Нет, но если это объект или массив, то содержимое можно изменять.

**В: Когда использовать `let`, а когда `const`?**
О: По умолчанию используй `const`, а `let` только когда значение точно будет изменяться.

**В: Что такое `undefined`?**
О: Это значение по умолчанию для переменных, которым не задано значение.

**В: Зачем нужны шаблонные строки?**
О: Они позволяют легко вставлять переменные в текст и создавать многострочные строки.
